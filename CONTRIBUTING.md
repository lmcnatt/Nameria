# Contributing to Nameria

Thank you for your interest in the Nameria D&D Species Portal!

## Academic Project Notice

This is an individual academic project for IS 531 at BYU. As such, direct contributions are not being accepted during the active course period.

However, you're welcome to:
- Fork the project for your own learning
- Report issues or bugs
- Suggest improvements
- Use this as a template for your own serverless projects

## After Course Completion

After the course ends (December 2025), this project may be opened for contributions. Check back later!

## Using This Project

If you're using this project as a learning resource or template:

1. **Give Credit**: Please reference this project if you use significant portions of the code
2. **Modify It**: Feel free to adapt it for your own use case
3. **Share Your Experience**: Let me know if this helped you learn serverless architecture!

## Reporting Issues

If you find a bug or have a suggestion:
1. Check if an issue already exists
2. Create a new issue with:
   - Clear description
   - Steps to reproduce (for bugs)
   - Expected vs actual behavior
   - Your environment (AWS region, Node version, etc.)

## Questions?

For questions about the architecture or implementation:
- Review the documentation in `/docs`
- Check the README.md
- Open a discussion on GitHub

## Code of Conduct

Please be respectful and constructive in all interactions.

---

**Note**: This is a learning project. While the code works and demonstrates serverless concepts, it's optimized for education rather than production use at scale.
